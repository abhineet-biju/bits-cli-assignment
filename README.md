# Graded Lab Assignment: Modules 1-4

This repository is prepared for the BITS Pilani Command Line Interfaces and Scripting graded lab assignment.

## Submission Checklist

- [ ] Repository is public on GitHub.
- [ ] `Question1` through `Question5` folders are present.
- [ ] Each question folder contains commands, outputs, screenshots, files created during the lab, and explanations.
- [ ] Outputs are collected from the Coursera Linux Lab environment.
- [ ] Screenshots show the actual command execution/output from the Coursera Linux Lab.
- [ ] Each command has a short observation/explanation.
- [ ] The GitHub repository URL is submitted in Coursera.

## Important

The command files and report templates are already written. Before final submission, run the commands in the Coursera Linux Lab and replace any placeholder output with your real lab output. Add screenshots into each `Screenshots` folder.

## Recommended Workflow In Coursera Lab

1. Open the Coursera Lab terminal.
2. Create or upload this repository folder into the lab.
3. Run each script:

```bash
bash scripts/run_question1.sh
bash scripts/run_question2.sh
bash scripts/run_question3.sh
bash scripts/run_question4.sh
bash scripts/run_question5.sh
```

4. Take screenshots for every important command/output and save them in the matching `Screenshots` folder.
5. Review each report and adjust usernames, paths, device names, and observations to match your real output.
6. Push the final folder to a public GitHub repository.
